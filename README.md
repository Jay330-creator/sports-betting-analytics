# sports-betting-analytics
## Overview
This project analyzes real-world sports betting data to evaluate profitability, win rate, and risk across different sports and betting strategies. Using SQL for data transformation and Tableau for visualization, the project identifies which betting approaches generate the highest returns and where inefficiencies exist.

![Dashboard](images/SportsTab.png)

## Dataset
The dataset consists of tracked betting activity across multiple sports, including bet type, odds, stake, result, and profit.
## Project Structure

- data/ → raw betting dataset  
- sql/ → SQL queries and table creation scripts  
- tableau/ → interactive dashboard file  
- images/ → dashboard and output visuals

- Betting frequency does not directly correlate with profitability, highlighting the importance of strategy over volume

- SQL + Tableau project analyzing sports betting performance, profitability, and risk through an interactive dashboard
## Key Insights
- NFL generated the highest total profit despite not having the highest win rate, indicating higher-value opportunities
- MLB achieved the highest win rate but produced lower total profit, suggesting lower return per bet
- Spread bets contributed the most to overall profitability across all sports
- Higher-risk odds ranges showed greater volatility but also the potential for larger returns
- Betting volume alone did not correlate with profitability, emphasizing the importance of strategy over frequency
## Impact
This analysis demonstrates how data-driven decision making can improve betting strategy by identifying high-performing bet types and optimal risk levels.
